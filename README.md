# fortaleza-rents-dataset

O dataset está disponível [no google colab clicando aqui.](https://colab.research.google.com/drive/1Q586Bez_jOmReF9tHmZl6-lCZybyDlrJ?usp=sharing)

## Instruções para executar localmente na sua máquina

0. Clone o repositório e entre na pasta do repositório

1. Criar virtualenv com `python3 -m venv venv`

2. Ativar virtualenv com `source venv/bin/activate`

3. Instalar dependências com `pip install -r requirements.txt`

4. Executar jupyter notebook com `jupyter notebook`
